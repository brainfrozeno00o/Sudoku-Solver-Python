1) Make sure you have 'pip' installed first. Check out this article on how to install pip: https://phoenixnap.com/kb/install-pip-windows
2) When 'pip' is installed, install the PyInstatller by typing "pip install pyinstaller".
3) Once PyInstaller is installed in your system, run the make-exe batch file. You can do this in the command line by simply typing "make-exe".
4) From there, you can now run SudokuSolver-Python.exe.

Enjoy!
